# Nottie - Save highlighted text as notes with page context

A simple Chrome extension to save highlighted text from any webpage as searchable notes.

## Features

- **Save Highlighted Text**: Select any text on a webpage and save it as a note
- **Auto-tagging**: Notes are automatically tagged with the page title and URL
- **Custom Tags**: Add your own tags to organize notes
- **Search**: Find saved notes by keyword, tag, title, or URL
- **Clean UI**: Beautiful, intuitive interface
- **Local Storage**: All notes stored locally in your browser
- **Export**: All notes are exportable in .txt format

## How to Use

### Saving a Note
1. Highlight any text on a webpage
2. Click the extension icon in your toolbar
3. The selected text will appear in the popup
4. (Optional) Add custom tags separated by commas
5. Click **Save Note**
6. You'll see a "Saved!" confirmation

### Searching Notes
1. Click the extension icon
2. Go to the **Search** tab
3. Enter a keyword to search
4. Click the search button or press Enter
5. Click on any note to see full details

### Managing Notes
- **Show All Notes**: View all saved notes
- **Clear All**: Delete all notes (with confirmation)
- **Delete Individual**: Click a note, then click Delete in the modal
- **Copy Text**: Copy the note text to clipboard
- **Open Source**: Open the original webpage
- **Export**: Save the note somewhere in your computer

## Privacy
All notes are stored locally in your browser using Chrome's storage API. No data is sent to any external servers.
